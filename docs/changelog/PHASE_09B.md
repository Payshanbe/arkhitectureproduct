# Phase 09B - Project Detail Editorial Polish

## Scope

Refined the approved Project Detail foundation without changing routing, CMS queries, page sections, or motion behavior.

## Changed

- Improved statement readability by splitting long editorial descriptions into natural paragraphs.
- Reduced statement scale slightly and increased line height for calmer reading.
- Increased statement and feature-image breathing room.
- Refined materials into a numbered editorial text list.
- Expanded gallery rhythm with a four-step image sequence:
  - full-width landscape
  - contained landscape
  - portrait-feeling image
  - offset landscape
- Improved caption spacing and hierarchy while keeping captions subtle.

## Not Changed

- No routing changes.
- No CMS query changes.
- No sliders, lightbox, sticky sections, image zoom, or new animations.
- Existing reveal and image fade behavior remains unchanged.

Run:

```bash
npm run lint
npm run build
npm run dev
```
