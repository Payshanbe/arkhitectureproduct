# 17 — Implementation Roadmap

## Purpose

This roadmap defines the complete implementation order of the project.

The goal is to build the website incrementally while maintaining quality, consistency, and stability.

Never skip phases.

Each phase builds on the previous one.

---

# Phase 01 — Project Foundation

Goal:

Create the technical foundation.

Tasks:

* Create Next.js project
* Configure TypeScript
* Configure Tailwind CSS
* Configure ESLint
* Configure Prettier
* Configure aliases
* Configure environment variables
* Configure folder structure
* Configure next/font
* Configure global styles

Deliverable:

A clean production-ready project foundation.

---

---

# Definition of Done

A phase is complete only when:

* functionality works;
* responsive behavior is verified;
* accessibility is verified;
* performance is acceptable;
* documentation remains up to date.

Never move to the next phase with unfinished work.

---

# Final Principle

Build slowly.

Build correctly.

Quality is always more important than speed.
