## Philosophy

The website is not a collection of pages.

It is a continuous visual journey through architecture, space, materials, light, and atmosphere.

Every page should naturally lead to the next one.

The user should feel like they are walking through a carefully designed architectural exhibition.

# Design Rule

Every page should feel like part of one continuous architectural experience.

Transitions between pages should feel like moving between rooms inside the same building.

Never make pages feel disconnected.
