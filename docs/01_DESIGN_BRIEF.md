# 01 — Design Brief

## Project Type

Premium website for an architecture and interior design agency.

This is not a standard marketing landing page.
The website should feel like a digital portfolio, an editorial presentation, and a cinematic architectural experience.

## Core Direction

The visual direction is:

* Quiet luxury
* Editorial minimalism
* Cinematic presentation
* Architecture-first layout
* Image-driven storytelling
* Smooth and restrained motion

The website should feel calm, premium, precise, spacious, and confident.

## Main Goal

The goal is to present the agency through atmosphere, projects, photography, typography, and spatial rhythm.

The website should not aggressively sell services.
It should build trust through taste, visual discipline, and high-quality project presentation.

## Main Principles

1. Photography is the main content.
2. Typography supports the atmosphere.
3. Whitespace creates premium feeling.
4. Motion should be slow, calm, and intentional.
5. Navigation should be minimal.
6. The site should feel custom, not template-based.
7. Every section should have strong composition.
8. The experience should be smooth on desktop and stable on mobile.

## Target Feeling

The site should feel like:

* a luxury architecture magazine;
* a high-end interior portfolio;
* a cinematic project archive;
* a digital extension of an architectural space.

## What To Avoid

Avoid:

* generic SaaS-style landing page blocks;
* colorful gradients;
* cheap hover effects;
* too many CTA buttons;
* icons with “benefits”;
* stock-style sections;
* aggressive sales copy;
* too much animation;
* bouncy or elastic motion;
* overloaded layouts;
* random decorative elements.

## Website Personality

The website should be:

* calm;
* elegant;
* minimal;
* serious;
* visual;
* premium;
* editorial;
* architectural;
* cinematic.

## Content Priority

The priority order is:

1. Project images
2. Project names
3. Studio philosophy
4. Project metadata
5. Services
6. Contact

## Recommended Pages

The website should include:

* Home
* Work / Projects
* Project Detail
* Studio / About
* Contact

Optional:

* Journal
* Press
* Services

## Technical Stack

The website will be built with:

* Next.js
* React
* TypeScript
* Lenis for smooth scrolling
* GSAP for animations
* ScrollTrigger for scroll-based animation
* CSS Modules, Tailwind CSS, or a clean global design system depending on project setup

## Motion Direction

Use only a few signature animations:

* smooth scroll with Lenis;
* image reveal using clip-path or mask;
* subtle text reveal on important headings;
* scroll-triggered section reveals;
* restrained parallax only where it adds depth;
* fullscreen menu animation;
* optional pinned section for one important storytelling moment.

Motion should feel slow, expensive, and controlled.

## Visual Direction Summary

The website should be built around large images, strong typography, whitespace, asymmetry, and subtle motion.

The result should feel closer to Awwwards-level architecture websites than to a normal agency landing page.
